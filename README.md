## Install

```sh
yarn add git-tag-cli
```

## Usage

```
Usage

   $ git-tag-cli <command> <params>

   $ git-tag-cli sample <param>             # Uses the <PARAM>
   
 Examples

   $ git-tag-cli sample TEST                # Uses the TEST
   $ git-tag-cli sample YOLO                # Uses the YOLO
```
